# Cloud Infrastructure Exercise

## Overview
This project is a Java Object-Oriented Programming (OOP) exercise that simulates cloud infrastructure resources.

The project demonstrates:
- Interfaces
- Abstract Classes
- Inheritance
- Method Overriding
- Encapsulation
- Polymorphism

## Project Structure

- Monitorable.java – Interface for monitoring cloud resources.
- CloudResource.java – Abstract base class for cloud resources.
- ComputeNode.java – Represents a compute resource node.
- DatabaseNode.java – Represents a database resource node.
- DatacenterConfig.java – Configuration settings for the datacenter.
- Main.java – Demonstrates the functionality of the system.

## Features

- Power management for cloud resources.
- Health monitoring reports.
- Compute node resource allocation.
- Database node storage management.
- Uptime tracking.

## Technologies

- Java
- Object-Oriented Programming (OOP)
